# lib-dart_oop_complete.dart
# Program OOP Dart - 8 Konsep

Kumpulan demonstrasi konsep Object-Oriented Programming (OOP) di Dart.

## 📚 Konsep yang Dipelajari

1. **Class Mobil** - Constructor (Default, Named, dari JSON)
2. **Class Warna** - Const Constructor (Immutable Object)
3. **Class Titik** - Initializer List
4. **Class Person** - Private Field
5. **Class Person** - Getter (Method vs Syntax)
6. **Class Person** - Setter (Method vs Syntax)
7. **Class Produk** - Validasi dengan Getter & Setter
8. **Class User** - Login & Ganti Password

## 🚀 Cara Menjalankan

### Online (DartPad)
1. Buka [DartPad](https://dartpad.dev)
2. Copy kode dari `lib/dart_oop_complete.dart`
3. Paste dan klik Run

### Lokal (Dart SDK)
```bash
dart run lib/dart_oop_complete.dart
